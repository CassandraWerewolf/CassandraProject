Moderators Edit Game Page Users Guide


As a moderator you have the power to edit almost all of the data in our database about your game.


The things you can change are:

  • Title - Click on the Title

    • Please keep the title the same as what you called them game in the BGG thread.

  • Moderators - Click on the word Moderator

    • You can add another moderator if someone is co-moderating this game with you. Just make sure you select both names when you hit submit or you will no longer be a moderator and as soon as you hit refresh on the page you will no longer be able to edit it.

  • Dates - Click on the word Dates, or on the Dates themselves

    • The Date must be in the format yyyy-mm-dd.

    • Please enter an End Date after your game officially ends.

  • Status - Click on the word Status, or on the status level next to it.

    • If the game is a Sub-Thread this should NEVER change.

    • Once you set the game from "In Progress" to "Finished" all of the fields in the players table will then be viable to all viewers.

    • If you set a game to "Finished" don't forget to all add an End Date and a Winner.

  • Winner - Click on the word Winner, or on the space near it

    • Once the game has finished please select a general winner. If the game was not team based, or was not a good vs evil then please choose Other. You can then give a better description of the winner in the "Description" Section.

  • BGG Thread ID - Click on the words BGG Thread ID or on the thread ID.

    • Only change this if you are changing a game from a Sign-up status to a In Progress status and the Game is being played in a new thread. This should not change once the game is In Progress. If it does, you will need to change the page you are viewing in your browsers URL.

  • Sub-Threads - Click on the words Sub-Threads or the space next to it.

    • Use this if there are sub threads for the different teams to use to communicate.

    • You just need to enter the BGG thread id, and then you can go to its game page to edit everything else.

    • If you type the wrong number or something you can delete the sub-thread. Don't delete the sub-thread unless it was a mistake because you will delete everything associated with the sub-thread including any posts that have been retrieved from BGG.

  • Description - Click on the word Description or on the description

    • Type a short game description here. Do not use this space to put your rule set. You can use this area to mention highlights of the game. To format any text in this space you must use html tags.

  • Player Details - Click the pencil next to the players name

    • All of these fields will be invisible to everyone except the Moderator until the game is marked as "Finished". However, each individual player will be able to see their information, except for the comments in the "Comments" field. All fields will be viewable by everyone once the game is "Finished". So if a player has a hidden role, input it only in the Comment section until the game is "Finished".

    • Replacement - If a player is replaced after the game starts please mark it as a replacement rather than deleting the player and adding the new one. If you mess up on when the player was replaced, you can delete the replacement and add it again.

    • Role Name - If the player had a role with a special name enter it here. It is up to you if you want to add generic names like "villager".

    • Role Type - Please enter the player's role type. This will be used for generating statistics for the player as that type of player. Please only use generic role types. Don't add a role type to the list if your game is the only one to use it.

    • Team - Please enter the player's team. This will be used for generating statistics for the players win score. If this was not a good vs evil team game please use Other. If this was a multi-team game please only select the winning team for a players who's team actually won. "Other" is an exception to this rule. Other will not be used in generating statistics.

    • Comment - This is a place where you can make any private notes for yourself about a player. They will only be revealed once the game is finished. I like to put when a player was removed from the game in this section. To format any text in this space you must use html tags.

  • Add Player - Click on the header of the Players Column

    • This will allow you to add a player. Please do not use this to replace a player. Instead click on the player that is being replaced and add a replacement player under their entry.

  • Role Name - Click on the header of the Role Name Column

    • This will allow you to change all of the players Role Names at one time, rather than individually.

    • If the player had a role with a special name enter it here. It is up to you if you want to add generic names like "villager".

  • Role Type - Click on the header of the Role Type Column

    • This will allow you to change all of the players Role Types at one time, rather than individually.

    • Please enter the player's role type. This will be used for generating statistics for the player as that type of player. Please only use generic role types. Don't add a role type to the list if your game is the only one to use it.

  • Team - Click on the header of the Team Column

    • This will allow you to change all of the Players Team assignments at one time, rather than individually.

Please be patient if when you click it seems like things are taking a while. I tried to install an image to let you know things were working behind the scenes but I could never quite get it to work.
